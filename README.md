# Titanic Survival Analytics Dashboard

## Overview

The Titanic Survival Analytics Dashboard is an interactive data visualization project built with Streamlit. It explores the Titanic dataset to uncover patterns and factors that influenced passenger survival during the disaster.

The dashboard enables users to interactively analyze passenger demographics, survival outcomes, and relationships between key variables through clear and informative visualizations.

## Features

* Interactive dashboard built with Streamlit
* Survival statistics and key performance indicators (KPIs)
* Passenger distribution by gender and class
* Survival analysis by gender and passenger class
* Age distribution analysis
* Fare distribution comparison by survival status
* Interactive data table for passenger records
* Automated insights and recommendations

## Technologies Used

* Python
* Streamlit
* Pandas
* Plotly Express

## Key Findings

* Female passengers had significantly higher survival rates than male passengers.
* First-class passengers were more likely to survive than passengers in lower classes.
* Passengers who paid higher fares generally experienced better survival outcomes.
* Gender and passenger class were among the strongest factors associated with survival.

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd Analytical_project
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Running the Application

Start the Streamlit application:

```bash
streamlit run app.py
```

The dashboard will open in your web browser and provide an interactive environment for exploring Titanic passenger data.
