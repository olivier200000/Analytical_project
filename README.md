# Titanic Survival Analytics Dashboard

A short analysis dashboard built with Streamlit that explores Titanic passenger data.

## Description

This dashboard presents concise overview of Titanic survival patterns by examining passenger demographics and key factors affecting survival. It includes:

- Overall survival metrics and passenger counts
- Passenger distribution by class and gender
- Survival rates broken down by gender and passenger class
- Age distribution among passengers
- Fare distribution by survival status
- A full passenger data table
- High-level insights and recommendations

## Key Findings

- Female passengers had a higher survival rate than males.
- First-class passengers were more likely to survive than second- and third-class passengers.
- Higher ticket fares were generally associated with better survival outcomes.
- Passenger class and gender appear to be strong predictors of survival.
